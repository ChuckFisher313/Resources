# Bash commands #

## Directories and files ##

`mkdir [path] `

Create a directory. 

` ls ` 
 
List files and folders in the current directory. 

*Switches* ` -a ` (all) ` -ld * ` (One column)

` touch [filename] ` 

Creates a file in the current directory. 

` cd [path] `

Changes the current directory to the path entered.
